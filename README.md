# DigiKhet
Smart Kheti Ke Sathi

DigiKhet is a farmer-centric digital platform that integrates AI-enabled agricultural decision support with a direct digital marketplace to promote transparent, data-driven, and inclusive farming practices.

The repository is structured to clearly separate product functionality, machine learning modules, and evaluation documentation. This organization ensures clarity and ease of review for technical evaluators, academic reviewers, and governance or policy stakeholders.

---

## Documentation

All evaluation, design, and methodology-related documentation is organized under the `docs/` directory.

Project Overview  
Path: `docs/PROJECT_OVERVIEW.md`  
Description: High-level description of the problem statement, solution approach, system workflow, user interface walkthrough, and expected impact.

Machine Learning Methodology  
Path: `docs/ML_METHODOLOGY.md`  
Description: Detailed explanation of datasets used, data preprocessing, model architectures, training strategy, evaluation methodology, and governance alignment.

System Architecture  
Path: `docs/SYSTEM_ARCHITECTURE.md`  
Description: Overall system design, component interaction, data flow, and scalability considerations.

---

## Repository Structure (High-Level)

frontend/  
Contains the web-based user interface for farmers and buyers.

backend/  
Contains the API and service layer supporting platform functionality.

crop-prediction/  
Contains experimental and exploratory machine learning components.

crop-yield-prediction/  
Contains the core crop yield prediction pipeline and associated datasets and models.

Plant-disease_detection/  
Contains the plant disease identification module.

docs/  
Contains structured documentation for evaluation, system design, and methodology.
